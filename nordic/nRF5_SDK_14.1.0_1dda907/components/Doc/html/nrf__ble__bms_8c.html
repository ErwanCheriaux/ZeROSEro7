)

Handle authorization request events from the Queued Write module.

Parameters
[in]p_bmsBond Management Service structure.
[in]p_qwrQueued Write Structure.
[in]p_evtEvent received from the Queued Writes module.
Return values
BLE_GATT_STATUS_SUCCESSIf the received event is accepted.
BLE_BMS_OPCODE_OPERATION_FAILEDIf the received event is not relevant for any of this module's attributes.
BLE_BMS_OPCODE_NOT_SUPPORTEDIf the received opcode is not supported.
BLE_GATT_STATUS_ATTERR_INSUF_AUTHORIZATIONIf the application handler returns that the authorization code is not valid.
