Detailed Description

This module implements functions for the ANT+ common pages.

ANT+ common data pages define common data formats that can be used by any device on any ANT network. The ability to send and receive these common pages is defined by the transmission type of the ANT channel parameter.

Note that all unused pages in this section are not defined and therefore cannot be used.
