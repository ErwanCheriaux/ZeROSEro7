◆ NRF_MODULE_ENABLED

#define NRF_MODULE_ENABLED (   module)    ((defined(module ## _ENABLED) && (module ## _ENABLED)) ? 1 : 0)

Check if selected module is enabled.

This is save function for driver enable checking. Correct from Lint point of view (not using default of undefined value).

Usage:

#if NRF_MODULE_ENABLED(UART)
...
#endif
Parameters
moduleThe module name.
Return values
1The macro <module>_ENABLE is defined and is non-zero.
0The macro <module>_ENABLE is not defined or it equals zero.
Note
This macro intentionally does not implement second expansion level. The name of the module to be checked has to be given directly as a parameter. And given parameter would be connected with _ENABLED postfix directly without evaluating its value.
